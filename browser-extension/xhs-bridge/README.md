# 小红书运营台助手

这是本地开发版浏览器扩展，用于把当前浏览器里的小红书登录态同步到本机后端。

## 安装

1. 打开 Microsoft Edge 的扩展管理页：`edge://extensions/`，或 Chrome 的 `chrome://extensions/`。
2. 开启“开发人员模式”。
3. 点击“加载解压缩的扩展”。
4. 选择本目录：`browser-extension/xhs-bridge`。
5. 安装或更新后，刷新本地运营台页面和已打开的小红书页面。

## 使用

1. 在当前浏览器登录 `https://www.xiaohongshu.com/`。
2. 回到本地运营台，点击“检测助手”。
3. 检测成功后，点击“同步登录态”。
4. 也可以点击浏览器扩展图标，在弹窗里点击“同步当前浏览器登录态”。

Cookie 只会发送到本机 `http://127.0.0.1:8787`，不会发送到云端。

## 权限说明

| 权限 | 用途 |
| --- | --- |
| `cookies` | 读取小红书登录 Cookie |
| `tabs` | 查找或打开小红书标签页 |
| `scripting` | 从小红书页面上下文补充检测登录态候选字段 |
| `storage` | 预留扩展本地状态 |

`host_permissions` 只包含小红书域名和本机开发服务地址：`127.0.0.1:5173`、`localhost:5173`、`127.0.0.1:8787`。

## 排查

- 同步前确认当前浏览器 Profile 已登录小红书。
- 如果提示没有找到小红书页面，请先打开并刷新 `https://www.xiaohongshu.com/`。
- 如果提示缺少 `a1`，说明当前登录态没有提供 redbook 所需的签名 Cookie。请刷新小红书页面、退出后重新登录，或手动从浏览器开发者工具中确认是否存在 `a1`。
- 本机后端必须运行在 `http://127.0.0.1:8787`。
